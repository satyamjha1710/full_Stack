About Course

Front-end web development is everything involved in programming the user interface of a web application.Typicaly it refers to the Hyper text Markup Language(HTML), Cascading Style Sheets(CSS) and Java Script portion of web site production as opposed to the database or server side programming. It encompasses everything from buliding a simple page of html text to creating complex,responsive HTML5 websites designed to be accessed via various different browsers , devices and screen sizes.
